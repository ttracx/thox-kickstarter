## Red flags

| Signal | Threshold | Action |
|---|---|---|
| Wrong reward price | Any time | Pause promo, fix, note change publicly |
| Old product name appears | Any canonical doc or public page | Fix immediately, rerun validator |
| Generated image looks like final production photo | Any reward tile | Replace or label as concept visual |
| Backer comment unanswered | More than 24 hours | Assign owner and reply |
| Security question unanswered | More than 4 hours | Tommy replies or drafts approved response |
| Payment setup issue | Any time | Stop launch or escalate immediately |
