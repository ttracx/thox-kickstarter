# Video Scene Prompts

Use this file to generate campaign graphics first, then create short video clips from the approved graphics.

```text
scene brief -> graphic prompt -> approve still -> image-to-video prompt -> edit into hero video
```

## Global graphic style

Add this style block to every graphic prompt:

```text
Create a premium Kickstarter campaign visual for THOX.ai private AI hardware. Dark navy and soft black background, emerald accent lighting, clean industrial design, realistic product-marketing composition, subtle privacy and security visual language, no clutter, no hype, no fake brand logos, no unreadable tiny text. Use crisp product silhouettes and clear hierarchy. Photorealistic where appropriate, but label as concept visual if not using real prototype photography. Brand line: Your AI. Your Data. Your Rules.
```

## Global image-to-video style

Add this style block to every image-to-video prompt:

```text
Animate the provided THOX.ai campaign graphic into a subtle premium hardware launch clip. Keep the device identity and composition consistent with the source image. Use slow camera movement, gentle emerald light pulses, small parallax depth, and restrained motion. Do not transform the product into a different object. Do not add new claims or new text. Export as a 4 to 6 second loop, 16:9, 1080p, smooth motion.
```
