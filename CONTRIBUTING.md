# Contributing

## Contribution Standard

Contributions must keep this repository production-oriented, documented, and legally clean.

Before proposing changes:

1. Read `README.md`, `docs/README.md`, `NOTICE.md`, and `SECURITY.md`.
2. Preserve THOX.ai LLC ownership notices and third-party license notices.
3. Add or update tests, validation commands, runbooks, and release notes when behavior changes.
4. Do not add secrets, private keys, credentials, or unapproved third-party assets.
