## Claim guardrails for the editor

Remove or rewrite any line that implies:

- ThoxKey runs large models standalone.
- ThoxMini Air is a high-end inference device.
- ThoxClip is a medical, emergency, safety, or regulated product.
- Generated product imagery is final production photography.
- Cloud services are required by default.
