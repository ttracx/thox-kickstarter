# MVP Catalog

Vertical-slice MVPs for the THOX.ai Kickstarter quick launch.

## MVP 1: Campaign Source of Truth

| Field | Value |
|---|---|
| Outcome | One canonical set of product names, prices, roles, and founder titles |
| AI slice | Claim-safety review and copy generation |
| Backend slice | Markdown docs and validation script |
| Frontend slice | README and Kickstarter paste-ready docs |
| Done when | `python3 scripts/validate_campaign.py` passes |

## MVP 2: Rewards and Pricing

| Field | Value |
|---|---|
| Outcome | Kickstarter reward tiers ready to paste |
| AI slice | Tier copy and FAQ support |
| Backend slice | `examples/reward-tier-import.csv` |
| Frontend slice | Reward cards on Kickstarter |
| Done when | Every retail, Kickstarter, and early-bird price matches source of truth |

## MVP 3: Kickstarter Story Page

| Field | Value |
|---|---|
| Outcome | Full Story page ready for Kickstarter |
| AI slice | Campaign narrative and visual prompts |
| Backend slice | `docs/KICKSTARTER_PAGE_COPY.md` |
| Frontend slice | Kickstarter Story editor |
| Done when | Story page is pasted, previewed, and approved on mobile |
