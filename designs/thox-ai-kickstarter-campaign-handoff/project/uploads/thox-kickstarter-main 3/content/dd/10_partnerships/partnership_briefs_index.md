## Status values

For the Status column, the user should use one of:

- INITIAL: brief drafted, no outreach yet.
- OUTREACH: outreach in flight, no reply.
- DIALOGUE: conversation underway, no commit.
- PILOT: pilot or beta agreement in motion.
- ACTIVE: live in production / public reference.
- SHELVED: explicitly de-prioritized for now.
