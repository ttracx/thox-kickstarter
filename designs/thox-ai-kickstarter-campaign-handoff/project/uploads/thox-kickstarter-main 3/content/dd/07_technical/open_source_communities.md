## Strategy

- Apache-2.0 across the contributed surface (see 03 IP open-source
  posture).
- Documentation hub at ttracx/thox-docs (Next.js 14 + Fumadocs).
- Tutorials, reference, and a quickstart-per-device pattern via
  ttracx/thox-quickstart.
- Welcoming-onboarding sequence for first-time contributors via
  thox-meta templates.
- Active maintainer presence on issues and PRs across the public
  surface; auto-merge enabled where appropriate, except for ttracx/Thox.ai
  marketing site (PR-only no-merge per policy).
