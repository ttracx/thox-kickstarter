# Glossary

Last updated: 2026-06-25
Status: DRAFT

---

- ADR - Architecture Decision Record. Per-decision short markdown doc
  capturing context, decision, and consequences. Used across the THOX
  portfolio per thox-meta.
- Apache-2.0 - the default outbound license across THOX repositories.
- BOM - Bill of materials. Per-SKU parts list with costs.
- Cap table - capitalization table. Who owns what equity in THOX.ai LLC.
- COGS - Cost of goods sold. Direct cost to produce and deliver one unit.
- D2C - Direct-to-consumer. Post-Kickstarter channel for ongoing sales.
- DCO - Developer Certificate of Origin. Lightweight contribution sign-off.
- DPA - Data Processing Addendum. B2B contract addendum for data handling.
- EDU - Educational pricing channel.
- EULA - End User License Agreement.
- FAST - Founder / Advisor Standard Template. Common advisor equity doc.
- GGUF - llama.cpp model format. Used for ThoxLLM and Gemma derivatives.
- HF - HuggingFace. THOX hosts the Thox-ai private model org there.
- LFS - Git Large File Storage.
- LiteRT-LM - on-device LLM runtime (Google). THOX maintains a pure-Rust
  rewrite as ttracx/thox-litert-lm.
- LLC - Limited Liability Company.
- LoRA - Low-Rank Adaptation. Fine-tuning technique used in
  thoxllm-factory.
- MagStack - magnetic-pogo clustering form factor used on ThoxMini Air.
- MCP - Model Context Protocol.
- MDX - Markdown extended with JSX. Used by thox-docs.
- MVP - in this codebase, a numbered milestone increment per the
  thoxos-kernel v1.2.0 plan (MVP-7..MVP-32). Not "minimum viable product."
- NDA - Non-Disclosure Agreement.
- Phase B / C - thoxllm-factory training phases.
- Phase D / E / F - thoxcore platform phases.
- QLoRA - Quantized LoRA. 4-bit fine-tuning variant.
- SAFE - Simple Agreement for Future Equity. Standard pre-priced
  investment instrument.
- SARIF - Static Analysis Results Interchange Format. Used by the AST
  scanner in thoxos-kernel.
- SBOM - Software Bill of Materials.
- SDK - Software Development Kit.
- SKU - Stock Keeping Unit. One distinct product variant.
- SLO - Service Level Objective.
- SoC - System on Chip.
- SPDX - Software Package Data Exchange. License identifier standard.
- ThoxClip - lowest-tier handheld THOX device. $39+ baseline placeholder.
- ThoxMini - small fixed THOX device. $69 placeholder.
- ThoxAir - small THOX device with MagStack clustering. $79 placeholder.
- ThoxNova - desktop-tier THOX device (LattePanda N100). $499 placeholder.
- ThoxOS - the supervisory layer over THOX devices.
- ThoxLLM - the THOX-owned base language model family.
- ThoxMicro-125M - the first published THOX base LM.
- THOX.ai LLC - the operating entity. State of formation under
  reconciliation.
- TBD - to be determined. Used as a placeholder in this packet.
- USPTO TESS - US Patent and Trademark Office trademark search system.

## Disclaimer

Glossary is a working reference. Definitions are simplified for
investor briefing and may abbreviate technical precision.
