## Edge cases

- thox-portable-agent and thoxymicro carry MIT because they fork
  MIT-licensed upstream projects. Outbound MIT is the courteous match.
- Brand assets (logos, marks, color palette) in thox-brand-vault remain
  proprietary by USE even though the repository file metadata is
  Apache-2.0.
- ThoxOS Sandbox is a branded fork of alibaba/OpenSandbox; upstream
  Apache-2.0 is preserved.

## Tooling

- thox-meta enforces SPDX headers and CI lint.
- License roll-up can be regenerated by walking ttracx/ via `gh repo list`
  and reading each repo's LICENSE file.

## Disclaimer

Outbound license posture only. Inbound dependency license review (transitive
GPL, AGPL, or copyleft contamination) requires separate audit and is not
covered by this packet.
