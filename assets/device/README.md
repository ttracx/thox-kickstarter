# Device renders

Studio product renders for each THOX device in the standard colorway set
(**Matte Black · Space Gray · Arctic White**). Use these as the canonical
device imagery for social art, decks, and the Kickstarter Story page until
EVT photography lands (see `docs/LAUNCH_CHECKLIST.md`).

Brand tokens and typography are the same as [`../README.md`](../README.md).
